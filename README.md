# Readdit for Reddit
A Reddit client for Elementary OS. (Still a work in progress)

![sample](https://github.com/xchan14/readdit/blob/master/docs/readme-image.png?raw=true)

Dependencies:
- gtk+-3.0
- granite
- gee-0.8'
- libsoup-2.4
- json-glib-1.0
- gio-2.0
- clutter-1.0
- clutter-gst-3.0
- clutter-gtk-1.0
- webkit2gtk-4.0

To build and run:

`cd build`

`sudo ninja install`

`com.github.xchan14.readdit`

To uninstall:

`cd build`

`sudo ninja uninstall`
